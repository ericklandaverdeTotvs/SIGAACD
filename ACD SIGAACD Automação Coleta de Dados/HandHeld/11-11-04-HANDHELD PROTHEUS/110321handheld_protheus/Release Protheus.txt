30/11/2007

- Implementado serviço de Metas (função HHEXPSCT)
- Implementado serviço de Consumo (função XEXPHCN)
- Criado parâmetro para alterar a data de emissão dos pedidos no handheld para a data de entrada destes pedidos no Protheus (se MV_HHDTSC5 = .T. a data do pedido sera trocada)
- Criado parâmetro para que os serviços genéricos sejam executados somente uma vez por dia, quando o JOB for ativado (Para ativar, preencher o parâmetro MV_HHSRGEN com a data atual)
- Implementado sistema para liberação automatica de usuarios bloquados por inconsistências durante o JOB, se for habilitado, cada vez que for encontrado um usuario bloqueado, o sistema ira recriar a base do mesmo, e liberá-lo para novas conexões (informar HHLAUTO = 1 no mp8srv.ini)
- Implementado mensagens de status no monitor do Mobile, informando pontualmente quantos usuários estão em sincronismo, em JOB, processando importação, bloqueados, etc
- Criado parâmetro para não exportar clientes bloqueados (MV_SFA1BLQ = N não exporta)
- Criado parâmetro para não exportar produtos bloqueados (MV_SFB1BLQ = N não exporta)
